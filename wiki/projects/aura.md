# PROJECT: AURA
> Field-intelligence assistant & Android launcher.

---

## OVERVIEW
Aura is an AI-driven Android launcher and field assistant. It integrates real-time Vision (Gemini), STT, and TTS with a local semantic memory system.

## KEY FEATURES
- **Multimodal:** Vision + Voice.
- **Serverless Fallback:** 35+ LLM providers managed via `ai-api-tools`.
- **Semantic Memory:** Built using SQLite/pgvector.
- **Native Android:** Kotlin/Jetpack Compose.
